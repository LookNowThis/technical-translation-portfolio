# Software Manual Sample

## English → Lithuanian

**Instruction:**  
Click **System Settings** to configure network access, security permissions, and application behavior.

**Translation:**  
Spustelėkite **Sistemos nustatymai**, kad sukonfigūruotumėte tinklo prieigą, saugos leidimus ir programos veikimą.

---

## Lithuanian → English

**Original:**  
Norėdami atnaujinti programą, pasirinkite **Pagalba → Tikrinti, ar yra naujinimų**.

**Translation:**  
To update the software, select **Help → Check for updates**.
