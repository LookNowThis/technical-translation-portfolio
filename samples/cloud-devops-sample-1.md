# Cloud / DevOps Sample

## English → Lithuanian

**Instruction:**  
Deploy the service using the provided Docker image and ensure that environment variables are correctly configured before starting the container.

**Translation:**  
Diekite paslaugą naudodami pateiktą „Docker“ atvaizdą ir įsitikinkite, kad prieš paleidžiant konteinerį aplinkos kintamieji yra tinkamai sukonfigūruoti.

---

## Lithuanian → English

**Original:**  
Norėdami atlikti sistemos diegimą debesijoje, paleiskite automatizuotą CI/CD procesą ir patikrinkite, ar visi testai praėjo sėkmingai.

**Translation:**  
To deploy the system in the cloud, run the automated CI/CD pipeline and verify that all tests have passed successfully.
